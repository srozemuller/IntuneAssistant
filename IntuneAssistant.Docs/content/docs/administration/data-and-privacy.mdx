# Data & Privacy Policy

Intune Assistant is built with privacy by design. Our core principle is that **your data stays in your tenant** - we process everything in memory and store only the absolute minimum required for licensing and identification.

## Our Privacy Commitment

<Callout type="success" title="Zero Data Extraction">
    **No data leaves your tenant.** All Intune and Graph API data is processed in memory only and never stored or transmitted to external systems.
</Callout>

### Data Flow Principles

1. **Request Processing**
   - API calls made directly to your tenant's Graph API
   - Data retrieved and processed in memory only
   - Results displayed immediately to your browser

2. **No Persistence**
   - API responses processed and immediately discarded
   - No database storage of any tenant-specific data
   - No caching beyond browser session storage

3. **Session Management**
   - Authentication tokens stored only in browser session
   - Session data cleared automatically on logout
   - No server-side session persistence

<Callout type="info" title="Browser Session Storage">
    We use browser session storage for temporary data like UI preferences and navigation state. This data is automatically cleared when you close your browser or log out.
</Callout>

## Data Security Measures

### Technical Safeguards

- **HTTPS Encryption** - All communications encrypted in transit
- **No Database Storage** - No persistent storage of tenant data
- **Memory-Only Processing** - All operations performed in volatile memory
- **Automatic Cleanup** - Memory cleared after each operation

## Frequently Asked Questions

### General Privacy Questions

**Q: Where is my data stored?**
A: Your data is never stored outside your tenant. We only store tenant ID and domain for licensing purposes. All other data remains in your Microsoft tenant.

**Q: Can you see my Intune configurations?**
A: No, we never store or log your Intune data. We only process it in memory to display it to you, then immediately discard it.

**Q: What happens to my data if I stop using Intune Assistant?**
A: Only your tenant ID and domain are stored for licensing. There's no other data to delete. You can request removal of licensing data by contacting support.

### Technical Questions

**Q: How do you handle authentication tokens?**
A: Authentication tokens are stored only in your browser's session storage and are never transmitted to our servers or stored persistently.

**Q: Is my data encrypted?**
A: All communication is encrypted via HTTPS. Since we don't store your data, there's no data at rest to encrypt.

**Q: Can other users see my data?**
A: No, all data processing is isolated per user session. No data is shared between users or stored centrally.
