## What you can do

- **Add** assignments (without touching existing ones)
- **Remove** specific assignments (without affecting others)
- **Replace** all assignments for a policy (clean migrations)
- Apply **device filters** (include/exclude) where supported

---

## Quick start

1. Go to **Extensions → Assignment Manager**
2. Download the CSV template
3. Upload your CSV
4. Review the compare results
5. Select changes and run the migration
6. Validate and export the results

<Callout type="tip" title="Use Git for your CSVs">
    Treat the CSV as “infrastructure as code”: commit it, review changes, and run predictable migrations.
</Callout>
